Meanscript is a multi-platform scripting language for data saving and transfer, and for programming as an interpreted programming language. Data can be represented in human-readable script format and compact binary bytecode. Featuring

  • Support C#, C++, Java, and JavaScript.
  • Meanscript compiler translates text script to bytecode (binary format)
  • Generate Meanscript data and byte code from your source code
  • Meanscript interpreter reads and executes bytecode by iterative, non-blocking execution
  • Read and write values from/to Meanscript variables and data structures
  • Call Meascript functions from your source code, or your callbacks from Meanscript.
  • Basic data types: integer, string, boolean.
  • Define data structures (like struct and class in other languages)
  • Define your own, additional data types from your source code
  • Built-in data structures: array, list, map (dictionary).
  • Program basic functions with arguments, return values, conditions ("if"), loops ("while"), etc.

Meanscript's design fundamentals

  • Practical. Many ways to use it. Both convenient and efficient.
  • Minimal language core and syntax, easy to write and read.
  • Easy to expand. Add your own data types and callbacks.
  • Easy to set up, stand-alone. Add Meanscript source code to your project and start scripting. Minimal dependencies to a platform and other libraries.
  • Convenient. Simple interface, easy to debug, and strong typing and type checking to avoid elusive bugs. Smooth crashing by exception hangling and with informative error messages.
  • Iterative, non-blocking script execution.

You can access Meanscript data from your source code. For example compile the next script to a binary file called example.bin. 

text name: "John"
int age: 38
Read and access data from your scource code, for example (in pseudo-Java) 
Meanscript ms = Meanscript.compile("example.bin");
String name = ms.getText("name");
int age = ms.getInt("age");

Auto-generated structures

Define data structure with generated API for your source code, e.g. 
struct Person [text name, int age]
Generate API with a Meanscript generation tool. In pseudo-Java: 
namespace MyMS {
  class Person {
    public Person();
    
    void    set_name(String name)  { ... }
    String  get_name()             { ... }
    void    set_age(int age)       { ... }
    int     get_age()              { ... }
  }
}
Create data using the new struct Person. 
Person boss: ["John", 38]
Person janitor: ["Jack", 65]
Access data from your source code. 
MyMS.Person bossData ms.getData("boss"); // auto-generated class

print(bossData.get_name() + " is the boss"); // Prints "John is the boss".

Built-in data structures

Create data using data strutures. For example create a list of Person type of items, using built-in Meascript list: 
list [Person] personList
personList.add ["John", 38]
personList.add ["Jane", 57]
// ...
personList.add ["Jack", 45]
Access data from your source code. 
MS.MList list = ms.getList("personList"); // Class MS.MList would be part of Meanscript core
Person second = list.getAt(1);
Person last = list.getLast();
print(second.get_name() + " is the 2nd"); // prints "Jane is the 2nd"

Data synchronization

Keep remote data updated by sending modifications (diff) instead of all data. For example, modify your data: 
MS.MList list = ms.getList("personList");
list.first.setText("name", "Carl");
